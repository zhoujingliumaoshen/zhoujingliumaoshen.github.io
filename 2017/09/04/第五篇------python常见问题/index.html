第五篇------python常见问题

发表于 2017-09-04

1. python3中，字典、元组不能直接索引！

如需索引字典、元组，必须先转化为列表形式，例如：firstStr = myTree.keys()[0]转化为 firstStr = list(myTree.keys())[0]

2. 列表、数组的区别！

列表
dataMat = [[1,2,3],[4,5,6],7,[8,9]]
数组 dataArr = array(dataMat)
dataArr = [[1 2 3]
           [4 5 6]
           7
           [8 9]]

3. weights = ones(n)表示什么？

表示生成一个行向量(数组)，n个元素，元素值全为1。
weights = [1. 1. 1.   ...   1.]
相当于  (1,1,1,   ...  ,1)

4.列表外面不能直接乘以浮点数！把列表转换成数组之后就可以了！

#####

1）  >>>alpha = 0.01
     >>>list1 = [2,2,5]
     >>>print(alpha*list1)
     TypeError: can't multiply sequence by non-int of type 'float'
2）  >>>alpha = 3
     >>>list1 = [2,2,5]
     >>>print(alpha*list1)
     [2, 2, 5, 2, 2, 5, 2, 2, 5]
3）  >>>alpha = 0.01
     >>>Arr1 = array(list1)
     >>>Arr1
     [2 2 5]
     >>>print(alpha*Arr1)
     [ 0.02  0.02  0.05]

5.多维数组

>>> b = np.array([(1.5,2,3), (4,5,6)])
>>> b
array([[ 1.5,  2. ,  3. ],
       [ 4. ,  5. ,  6. ]])
