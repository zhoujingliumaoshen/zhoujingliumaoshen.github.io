hexo-next主题下发布新文章只需三步

发表于 2017-08-26

打开命令框

第一步：$ hexo new “文章名”

这时你会发现/source/_posts目录下新建了一个文件叫做 文章名.md，编辑该文件里面的内容并保存。（我们可以预先在markdown中提前编辑好文章内容，然后直接贴到 .md文件中）

第二步：$ hexo generate

第三步：$ hexo deploy

此时打开你的博客，就能看见一篇新的文章已经建好了，是不是很容易呢？^_^
